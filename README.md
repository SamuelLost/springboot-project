# Como executar o projeto

## Back end
Pré-requisitos: Java 17

```bash
# clonar repositório
git clone https://github.com/SamuelLost/springboot-project.git

# entrar na pasta do projeto back end
cd springboot-project

# executar o projeto
./mvnw spring-boot:run
```
